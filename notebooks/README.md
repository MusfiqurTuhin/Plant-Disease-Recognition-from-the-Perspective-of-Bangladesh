# Notebooks

This directory contains the Jupyter notebooks used for training and evaluating the deep learning models.

## Notebooks Overview

1.  **01_ConvNeXtBase_Transfer_Learning.ipynb**: Implementation of Transfer Learning using the ConvNeXtBase architecture.
2.  **02_ResNet101V2_Transfer_Learning.ipynb**: Implementation of Transfer Learning using ResNet101V2.
3.  **03_ResNet152V2_Transfer_Learning.ipynb**: Implementation of Transfer Learning using ResNet152V2.
4.  **04_VGG16_Transfer_Learning.ipynb**: Implementation of Transfer Learning using VGG16.
5.  **05_VGG19_Transfer_Learning.ipynb**: Implementation of Transfer Learning using VGG19.
6.  **06_Xception_Transfer_Learning.ipynb**: Implementation of Transfer Learning using Xception.
7.  **07_Ensemble_Model.ipynb**: Implementation of the weighted average ensemble model combining Xception, VGG19, and ResNet152V2.

## Usage

To run any of these notebooks, ensure you have the dataset located in `../Dataset/` relative to this directory (or update the paths in the notebooks accordingly).

```bash
jupyter notebook 01_ConvNeXtBase_Transfer_Learning.ipynb
```
